# logica-algoritmos-python
Repositório com projetos desenvolvidos durante meus estudos de lógica de programação e algoritmos utilizando Python.


## Sobre
Este repositório reúne aplicações simples em terminal (CLI) com foco na prática dos principais conceitos da programação.



## Projetos

### Calculadora de Plano de Saúde

Cálculo de mensalidade com base na idade do cliente e valor do plano.

### Sistema de Pedidos - Pizzaria

Simulação de pedidos com escolha de sabor, tamanho e cálculo do valor total.

### Simulador de Venda de Madeira

Cálculo de vendas com base em tipo de madeira, quantidade, desconto e transporte.

### Gerenciador de Contatos

Sistema de cadastro, consulta e remoção de contatos via terminal.

## Conceitos aplicados

* Estruturas condicionais (if, elif, else)
* Estruturas de repetição (while)
* Funções
* Listas e dicionários
* Tratamento de erros (try/except)

## Tecnologias

* Python


## Objetivo

Consolidar conhecimentos em lógica de programação e construir uma base sólida para desenvolvimento de sistemas mais complexos.
